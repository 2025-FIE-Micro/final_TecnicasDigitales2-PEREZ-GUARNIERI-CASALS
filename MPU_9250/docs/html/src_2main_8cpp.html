Descripción detallada

Lectura del sensor MPU-9250 y transmisión de datos por puerto serial en formato JSON.

El programa mide aceleración (g), velocidad angular (°/s) y campo magnético (µT), usando un ESP8266 y el sensor MPU-9250. Los datos se imprimen en formato JSON por serial.
